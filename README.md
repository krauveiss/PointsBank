# Points Bank

Серверное приложение на нативном PHP для работы с кошельками и переводами между пользователями.

##  О проекте

Это pet-проект, который я делал для того, чтобы разобраться, как работают веб-фреймворки "под капотом". В проекте нет Laravel или других библиотек — вся логика реализована вручную: роутинг, обработка запросов, работа с сессиями и базой данных.

---

##  Функциональность

- Регистрация и авторизация пользователей
- Создание кошельков
- Просмотр баланса
- Переводы между кошельками
- История транзакций

---

##  Архитектура

Реализована простая MVC-структура приложения:

- Router — обработка URL и маршрутизация
- Controllers — логика обработки запросов
- Models / DB слой — работа с базой данных через PDO
- Views — отображение данных

Также реализована работа с сессиями и базовая структура приложения без использования сторонних фреймворков.

---

##  База данных

- Спроектирована структура базы данных
- Используется MySQL
- Работа с БД через PDO
- Реализованы транзакции для операций перевода

---

##  Стек

- PHP 8.x
- MySQL
- PDO
