# Points Bank

Простое серверное приложение на нативном PHP для работы с кошельками и переводами между пользователями.

##  О проекте

Этот проект я делал, чтобы разобраться, как работают веб-фреймворки "под капотом". Здесь нет Laravel или других фреймворков,вся логика написана вручную: роутинг, обработка запросов, работа с сессиями и базой данных.

---

##  Функциональность

- Регистрация и авторизация пользователей
- Создание кошельков
- Просмотр баланса
- Переводы между кошельками
- История транзакций

---

##  Архитектура

Реализована простая MVC-структура:

- Router — обработка URL и маршрутизация
- Controllers — логика обработки запросов
- Models / DB слой — работа с базой данных через PDO
- Views — отображение

Также реализована работа с сессиями и базовая структура приложения без сторонних библиотек.

---

##  Работа с базой данных

- Используется MySQL
- Подключение через PDO
- Реализованы транзакции


---

##  Стек

- PHP 8.x
- MySQL
- PDO
